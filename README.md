# Microplastics Awareness Site

Basic HTML, CSS and vanilla JS static website about microplastic awareness, assigned as a civic education project in 2nd year of high school.

Live at: [roma476.github.io/Microplastics_Awarness_Site](https://roma476.github.io/Microplastics_Awarness_Site)

## About

Group project by the students of class 2C INFO at ITSOS. Each group researched a different topic related to plastic pollution. The entire codebase was written by one student (the repo owner); all other contributors handled research and content only. The result was assembled into a single static site hosted on GitHub Pages.

## Pages

- **Introduction to Plastic Pollution** — overview of the problem
- **Sources of Plastic Pollution** — how plastic enters the environment
- **The Fate of Discarded Plastics** — what happens after we throw plastic away
- **The Great Pacific Garbage Patch** — the largest accumulation of ocean plastic
- **The Microplastics Problem** — effects on marine life and human health
- **Solutions and Initiatives** — NGOs and projects fighting plastic pollution

## Game

Includes a small browser game — **Dodge the Microplastics** — built from scratch in vanilla JS, where the player dodges falling plastic particles.

## Tech Stack

- HTML, CSS, vanilla JavaScript — no frameworks or libraries
- Hosted on GitHub Pages
